# Tutorial: replicar la tienda de referencia en WordPress y WooCommerce

Este tutorial reúne los pasos principales para reproducir esta tienda de referencia:

- https://dev-noe-dam.pantheonsite.io/

El objetivo no es crear cualquier tienda genérica, sino construir una réplica de esa tienda usando WordPress, Pantheon, WooCommerce, Astra y Elementor.

La idea no es profundizar al máximo en cada pantalla, sino tener una visión ordenada del proceso completo.

## Fase 1: base técnica del sitio

### 1. Crear el sitio en Pantheon

Primero hay que crear el sitio web en Pantheon.

### 2. Instalar y configurar WordPress por primera vez

Después hay que completar la instalación inicial de WordPress y acceder al escritorio.

### 3. Instalar y activar el tema Astra

Una vez dentro de WordPress, conviene activar el tema que servirá como base visual del sitio.

### 4. Instalar el plugin WooCommerce

Después se instala WooCommerce para convertir el sitio en una tienda online.

## Fase 2: estructura del catálogo

### 5. Crear las categorías

Las categorías servirán para organizar los productos dentro de la tienda.

### 6. Crear las etiquetas

Las etiquetas permitirán agrupar productos de forma más flexible.

### 7. Crear los atributos

Los atributos permitirán describir productos y, en algunos casos, crear variaciones.

### 8. Crear las marcas

Las marcas permitirán identificar fabricantes o líneas comerciales.

## Fase 3: creación del catálogo

### 9. Crear los productos

En esta fase se deben crear:
- productos simples
- al menos algún producto variable

## Fase 4: navegación e identidad

### 10. Crear el menú principal

Hay que construir la navegación principal del sitio.

### 11. Añadir en el footer el texto del proyecto

En el pie de página debe aparecer este texto:

`Tienda ficticia para el módulo de Digitalización`

### 12. Añadir un icono del sitio como logo

Esto ayuda a dar identidad visual básica a la tienda.

## Fase 5: personalización visual

### 13. Personalizar la pantalla del catálogo

Conviene ajustar la forma en que se muestran los listados de productos.

### 14. Personalizar la página de producto

También conviene revisar y ajustar la presentación de cada ficha de producto.

### 15. Cambiar colores y tipografía del sitio

Esto permitirá adaptar la tienda a una identidad visual más coherente.

## Fase 6: páginas especiales

### 16. Instalar el plugin Elementor

Elementor servirá para construir páginas con mayor control visual.

### 17. Crear una landing page con Elementor

La landing page servirá como página promocional o de captación.

### 18. Crear la página Home

Después conviene crear una página de inicio propia.

### 19. Configurar Home como página inicial del sitio

Finalmente, hay que hacer que la portada del sitio sea la página `Home`.

## Idea clave

Replicar esta tienda no consiste solo en subir productos.

También implica:
- preparar la base técnica
- estructurar bien el catálogo
- organizar la navegación
- definir una identidad visual
- construir las páginas principales del sitio
