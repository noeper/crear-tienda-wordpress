# Reference: pantalla Menús en el personalizador

## Ruta

`Escritorio de WordPress -> Personalizar -> Menús`

## Para qué sirve esta pantalla

Permite:
- crear menús
- editar menús existentes
- asignar una ubicación al menú
- añadir y reorganizar elementos del menú

## Acción principal

Botón:
- `Crea un nuevo menú`

## Campos y opciones principales

### Nombre del menú

Permite escribir el nombre con el que se guardará el menú.

### Ubicación del menú

Permite asignar el menú a una ubicación del tema activa en el sitio.

En esta documentación, la ubicación usada es:
- `Primary Menu`

### Añadir elementos

Botón:
- `+ Añadir elemento`

Al pulsarlo, se abre un panel lateral con distintos tipos de contenido que pueden añadirse al menú.

## Tipos de elementos que pueden aparecer

Según la configuración del sitio, pueden aparecer opciones como:
- `Enlaces personalizados`
- `Páginas`
- `Entradas`
- `Productos`
- `Categorías`
- `Etiquetas`
- `Formatos`
- `Brands`
- `Product categories`
- `Product tags`
- `WooCommerce Endpoints`

## Comportamiento relevante

- Los elementos añadidos se incorporan al menú visible en el panel.
- Los elementos pueden reordenarse.
- Un elemento puede colocarse dentro de otro moviéndolo hacia la derecha.
- `Categorías` corresponde a las categorías de entradas de WordPress.
- `Product categories` corresponde a las categorías de producto de WooCommerce.
