# Reference: páginas creadas por WooCommerce

## Contexto

Durante la instalación o configuración inicial, WooCommerce puede crear páginas por defecto para la tienda.

## Páginas por defecto habituales

- `Shop`
- `Cart`
- `Checkout`
- `My Account`
- `Refund and Returns Policy`

## Qué hace cada una

### Shop

Es la página principal de catálogo de productos.

### Cart

Muestra el contenido del carrito.

### Checkout

Muestra el proceso de compra, incluyendo opciones como envío y pago.

### My Account

Muestra información relacionada con la cuenta del cliente, sus pedidos y otras opciones personales.

### Refund and Returns Policy

Es un borrador no publicado con texto de ejemplo para ayudar a redactar la política de devoluciones.

## Idea clave

Estas páginas muestran cómo WooCommerce añade una capa de ecommerce sobre WordPress mediante páginas y configuraciones nuevas que no estaban en la instalación base.
