# Reference: pantallas Etiquetas y Brands

## Pantalla Etiquetas

### Ruta

`Escritorio de WordPress -> Productos -> Etiquetas`

### Campos

- `Nombre`
- `Slug`
- `Descripción`

### Características

- Las etiquetas no son jerárquicas.
- Todas están al mismo nivel.

### Acción principal

Botón:
- `Add new tag`

## Pantalla Brands

### Ruta

`Escritorio de WordPress -> Productos -> Brands`

### Campos

- `Nombre`
- `Slug`
- `Parent Brand`
- `Descripción`
- `Thumbnail`

### Características

- Las marcas pueden ser jerárquicas.
- `Parent Brand` se usa para crear submarcas.

### Acción principal

Botón:
- `Add New Brand`
