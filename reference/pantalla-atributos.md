# Reference: pantalla Attributes

## Ruta

`Escritorio de WordPress -> Productos -> Attributes`

## Para qué sirve esta pantalla

Permite:
- crear atributos globales
- ver los atributos ya creados
- editar o eliminar atributos existentes

## Campos del formulario

### Name

Nombre visible del atributo.

Ejemplos:
- Talla
- Color

### Slug

Parte final de la URL del atributo.

Reglas:
- solo letras, números y guiones
- sin espacios

### Enable Archives?

Si esta opción se activa:
- WooCommerce crea una página para ese atributo
- permite navegar por ese atributo

Ejemplo:
- `/pa_color/rojo`

### Default sort order

Define cómo se ordenan los valores del atributo.

Opciones:
- `Custom ordering`
- `Name`
- `Name (numeric)`
- `Term ID`

## Acción principal

Botón:
- `Add attribute`
