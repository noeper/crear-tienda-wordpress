# Reference: pantalla Categorías

## Ruta

`Escritorio de WordPress -> Productos -> Categorías`

## Campos del formulario

### Nombre

Nombre visible de la categoría.

### Slug

Parte final de la URL.

Reglas:
- solo letras, números y guiones
- no usar espacios

### Categoría superior

Permite indicar si la categoría cuelga de otra categoría.

### Descripción

Texto breve sobre la categoría.

### Display type

Define qué se muestra cuando el usuario entra en la categoría.

Opciones:
- `Products`
- `Subcategories`
- `Both`
- `Default`

### Thumbnail

Imagen representativa de la categoría.

## Características del sistema

- Las categorías son jerárquicas.
- Un producto puede pertenecer a varias categorías.
- En este curso se usará una sola categoría principal por producto.

## Acción principal

Botón:
- `Add new category`
