# Reference: pantalla de edición de producto

## Ruta

`Escritorio de WordPress -> Productos -> Añadir nuevo producto`

## Bloques principales

La pantalla de edición de producto incluye estos bloques:
- `Product name`
- `Product description`
- `Product short description`
- `Product data`
- `Product image`
- `Product categories`
- `Brands`
- `Product tags`
- `Product gallery`
- `Publicar`

## Organización de bloques

Cada bloque tiene controles para:
- subir el bloque
- bajar el bloque
- plegar o desplegar su contenido

## Bloque Product data

Desde este bloque se define el tipo de producto.

Opciones mencionadas en esta documentación:
- `Simple product`
- `Variable product`

También contiene pestañas y zonas de trabajo como:
- `General`
- `Inventory`
- `Shipping`
- `Linked products`
- `Attributes`
- `Variations`

## Bloque Publicar

Opciones mencionadas:
- `Vista previa`
- `Guardar borrador`
- `Publicar`
- `Actualizar`

## Comportamiento relevante

- En productos simples, `Attributes` sirve como información adicional.
- En productos variables, `Attributes` puede usarse para crear variaciones.
- En productos variables, las variaciones se guardan con `Save changes`.
