# How-to: crear un atributo global

## Objetivo

Crear un atributo reutilizable en WooCommerce, como Talla o Color.

## Requisitos previos

- Tener claro el nombre del atributo.
- Saber si quieres activar archivos de atributo.
- Saber cómo quieres ordenar sus valores.

## Pasos

1. Ve a `Escritorio de WordPress -> Productos -> Attributes`.
2. En el formulario de creación, introduce el nombre del atributo en `Name`.
3. Define el `Slug` usando solo letras, números y guiones.
4. Decide si activas `Enable Archives?`.
5. En `Default sort order`, selecciona el criterio de orden.
6. Para este curso, usa `Custom ordering` si necesitas controlar manualmente el orden de los valores.
7. Pulsa `Add attribute`.

## Resultado esperado

El nuevo atributo aparece en el listado de atributos y ya puede configurarse con sus valores.
