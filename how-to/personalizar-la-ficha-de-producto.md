# How-to: personalizar la ficha de producto

## Objetivo

Ajustar cómo se muestra la página individual de un producto desde el personalizador de WordPress.

## Requisitos previos

- Tener activo un tema compatible con las opciones de WooCommerce en `Personalizar`.
- Tener WooCommerce instalado y activado.

## Ruta

`Escritorio de WordPress -> Apariencia -> Personalizar -> WooCommerce -> Single Product`

## Pasos

1. Ve a `Escritorio de WordPress -> Apariencia -> Personalizar`.
2. Se abrirá una nueva ventana con el sitio a la derecha y un panel lateral a la izquierda.
3. En ese panel, entra en `WooCommerce`.
4. Después entra en `Single Product`.
5. En este apartado puedes personalizar la página donde se muestra un producto concreto.
6. Revisa las opciones disponibles y prueba los cambios que quieras aplicar a esa ficha.
7. Observa en la vista previa cómo cambia la presentación del producto.
8. Cuando la ficha quede como quieres, pulsa `Publicar`.

## Resultado esperado

La página individual de producto muestra una presentación más ajustada a la identidad visual y a las necesidades del sitio.

## Importante

No hay que confundir:
- `Product Catalog`, que controla el listado general de productos
- `Single Product`, que controla la página de detalle de cada producto
