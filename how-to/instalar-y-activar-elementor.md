# How-to: instalar y activar Elementor

## Objetivo

Instalar y activar el plugin Elementor en WordPress.

## Requisitos previos

- Tener acceso al escritorio de WordPress.

## Ruta

`Escritorio de WordPress -> Plugins -> Añadir plugin`

## Pasos

1. Ve a `Escritorio de WordPress -> Plugins -> Añadir plugin`.
2. En la pantalla de plugins, localiza el campo `Buscar plugins`.
3. Escribe `Elementor`.
4. Revisa los resultados que aparecen.
5. Identifica el plugin correcto comprobando estos elementos:
   - el nombre `Elementor`
   - el icono con fondo rosa y la letra `E`
   - una descripción que indica que es un maquetador web con editor de arrastrar y soltar
6. Si necesitas apoyo visual, puedes comparar el icono con:
   - `assets/images/icon-elementor.gif`
7. Pulsa `Instalar`.
8. Cuando termine la instalación, pulsa `Activar`.

## Resultado esperado

- Elementor queda instalado en el sitio.
- Elementor pasa a estar activo.
- Las páginas ya pueden abrirse con la opción `Editar con Elementor`.

## Importante

No conviene identificar el plugin solo por una frase exacta de la descripción, porque ese texto puede cambiar con el tiempo.

Lo más seguro es confirmar a la vez:
- el nombre `Elementor`
- el icono rosa con la `E`
- y que el plugin corresponde al constructor visual de páginas
