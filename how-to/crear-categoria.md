# How-to: crear una categoría

## Objetivo

Crear una categoría para organizar productos dentro de la tienda.

## Requisitos previos

- Tener claro el nombre de la categoría.
- Saber si será una categoría principal o una subcategoría.

## Pasos

1. Ve a `Escritorio de WordPress -> Productos -> Categorías`.
2. Introduce el nombre en `Nombre`.
3. Define el `Slug` usando letras, números y guiones.
4. Si es subcategoría, selecciona la `Categoría superior`.
5. Añade una `Descripción` si procede.
6. En este curso, selecciona `Products` en `Display type`.
7. Si quieres, añade una imagen representativa.
8. Pulsa `Add new category`.

## Resultado esperado

La categoría aparece en el listado y ya puede asignarse a productos.
