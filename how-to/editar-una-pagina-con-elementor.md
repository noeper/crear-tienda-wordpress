# How-to: editar una página con Elementor

## Objetivo

Editar una página de WordPress usando Elementor.

## Requisitos previos

- Tener instalado y activado el plugin Elementor.
- Tener creada la página que se quiere editar.

## Ruta

`Escritorio de WordPress -> Páginas`

## Pasos

1. Ve a `Páginas`.
2. Localiza la página que quieres editar.
3. Pulsa `Editar con Elementor`.
4. Espera a que se abra la interfaz visual de Elementor.
5. Haz clic sobre el bloque o elemento que quieras modificar.
6. Cambia el contenido, el estilo o la disposición desde el panel lateral.
7. Repite el proceso con los elementos que necesites editar.
8. Cuando termines, pulsa el botón para guardar, publicar o actualizar los cambios.

## Qué se puede editar

Con Elementor se pueden editar elementos como:
- títulos
- textos
- imágenes
- botones
- columnas
- secciones

## Resultado esperado

La página queda actualizada visualmente con los cambios realizados en Elementor.
