# How-to: crear una marca

## Objetivo

Crear una marca para identificar el fabricante o línea comercial de un producto.

## Requisitos previos

- Tener claro el nombre de la marca.
- Saber si será una marca principal o una submarca.

## Pasos

1. Ve a `Escritorio de WordPress -> Productos -> Brands`.
2. Introduce el nombre de la marca.
3. Define el `Slug` usando solo letras, números y guiones.
4. Si es una submarca, selecciona `Parent Brand`.
5. Añade una `Descripción` si procede.
6. Si quieres, sube una imagen o logo en `Thumbnail`.
7. Pulsa `Add New Brand`.

## Resultado esperado

La nueva marca aparece en el listado y puede asignarse a productos.
