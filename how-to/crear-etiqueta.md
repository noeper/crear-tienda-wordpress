# How-to: crear una etiqueta

## Objetivo

Crear una etiqueta para agrupar productos de forma flexible.

## Requisitos previos

- Tener claro el nombre de la etiqueta.

## Pasos

1. Ve a `Escritorio de WordPress -> Productos -> Etiquetas`.
2. Introduce el nombre de la etiqueta.
3. Define el `Slug` sin espacios y usando solo letras, números y guiones.
4. Escribe una `Descripción` si aporta contexto.
5. Pulsa `Add new tag`.

## Resultado esperado

La etiqueta queda disponible para asignarla a productos.
