# How-to: crear la página Home con contenedores con imagen de fondo

## Objetivo

Construir en Elementor una estructura como la de la página `Home` de la tienda, basada en contenedores con imagen de fondo y con texto y botón superpuestos.

## Requisitos previos

- Tener instalado y activado el plugin Elementor.
- Tener creada la página `Home`.
- Tener preparadas las imágenes que se van a usar como fondo o saber de dónde obtenerlas.

## Ruta

`Escritorio de WordPress -> Páginas -> Home -> Edit with Elementor`

## Alcance de este tutorial

En este tutorial solo se explica cómo crear el bloque visual base:

- una imagen de fondo
- un texto superpuesto
- un botón superpuesto

Esa misma estructura se repite varias veces en la página `Home` de esta tienda.

## Pasos

1. Ve a la página `Home` y pulsa `Edit with Elementor`.
2. Dentro del maquetador, selecciona el contenedor que quieras usar para ese bloque.
3. En el panel lateral izquierdo, entra en la pestaña `Style`.
4. Localiza la opción `Background Type`.
5. En esa zona verás varios iconos, por ejemplo:
   - `Classic`
   - `Gradient`
   - `Video`
   - `Slideshow`
6. Para este caso, pulsa el icono `Slideshow`, representado con un grupo de imágenes.
7. Justo debajo aparecerá un cuadrado gris oscuro con un círculo `+` en el centro.
8. Haz clic en ese cuadrado y añade la imagen que quieras usar como fondo.

## Añadir el texto dentro del contenedor

1. Pulsa `Escape` para salir del foco actual.
2. En el panel lateral izquierdo, busca el bloque `Paragraph`.
3. Arrástralo al centro del contenedor.
4. Ajusta después el contenido y el estilo desde el panel lateral.

## Añadir el botón dentro del contenedor

1. Pulsa `Escape` si hace falta para volver al listado general de bloques.
2. Busca el bloque `Button`.
3. Arrástralo al mismo contenedor donde has colocado el texto.
4. Ajusta después el texto, el enlace y el estilo del botón.

## Si cuesta colocar bien los bloques

Si el texto o el botón no se ubican correctamente, conviene usar la opción `Structure`.

`Structure` está en la cabecera del maquetador de Elementor y permite ver la distribución interna de los bloques dentro de la página.

Esta vista ayuda a:

- comprobar en qué contenedor estás trabajando
- ver si el texto y el botón han quedado dentro del bloque correcto
- reorganizar los elementos con más precisión

## Repetir la estructura en la Home

En la página `Home` de esta tienda, esta estructura se repite cuatro veces.

Lo que cambia en cada caso es principalmente:

- la imagen de fondo
- el texto
- el botón

Las imágenes necesarias también están disponibles en el repositorio de contenido multimedia usado para crear la web.

## Resultado esperado

La página `Home` queda construida a partir de varios contenedores con imagen de fondo y con texto y botón superpuestos, siguiendo la misma lógica visual que la tienda de referencia.
