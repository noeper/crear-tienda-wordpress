# How-to: personalizar el catálogo de productos

## Objetivo

Ajustar cómo se muestra el catálogo de productos de WooCommerce desde el personalizador de WordPress.

## Requisitos previos

- Tener activo un tema compatible con las opciones de WooCommerce en `Personalizar`.
- Tener WooCommerce instalado y activado.

## Ruta

`Escritorio de WordPress -> Apariencia -> Personalizar -> WooCommerce -> Product Catalog`

## Pasos

1. Ve a `Escritorio de WordPress -> Apariencia -> Personalizar`.
2. Se abrirá una nueva ventana con el sitio a la derecha y un panel lateral a la izquierda.
3. En ese panel, entra en `WooCommerce`.
4. Después entra en `Product Catalog`.
5. Se abrirá un nuevo panel con dos pestañas:
   - `GENERAL`
   - `DESIGN`
6. Dentro de `GENERAL`, revisa las opciones disponibles para decidir cómo quieres mostrar el catálogo.
7. Ahí puedes ajustar aspectos como:
   - si se muestra o no el botón de compra
   - si se muestran o no las reseñas
   - si se muestran o no las categorías
   - cuántos productos quieres mostrar por línea
   - otras opciones visuales o funcionales del listado
8. Ve probando los cambios y observa cómo se reflejan en la vista previa.
9. Cuando el catálogo quede como quieres, pulsa `Publicar`.

## Resultado esperado

El catálogo de productos muestra una presentación más ajustada a la identidad visual y a las necesidades del sitio.

## Importante

En este apartado conviene experimentar.

Prueba todas las opciones que veas y compara su efecto en la vista previa antes de decidir la configuración final.
