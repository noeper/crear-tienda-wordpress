# How-to: instalar y configurar WordPress por primera vez

## Objetivo

Completar la instalación inicial de WordPress y acceder por primera vez a su escritorio de administración.

## Punto de partida

Este proceso empieza después de crear el sitio en Pantheon y entrar en `Dev admin`.

## Paso 1: elegir idioma

Lo primero que aparece es un listado de idiomas.

Selecciona el idioma con el que quieres usar WordPress.

## Paso 2: completar el formulario de instalación

En la siguiente pantalla aparece el formulario inicial de WordPress.

En él se indica algo parecido a:

- `¡Este es el famoso proceso de instalación de WordPress en cinco minutos!`

Debes completar estos campos:
- `Título del sitio`
- `Nombre de usuario`
- `Contraseña`
- `Tu correo electrónico`
- `Visibilidad en los motores de búsqueda`

### Qué poner en cada campo

#### Título del sitio

Escribe el nombre que tendrá tu sitio web.

#### Nombre de usuario

Escribe el nombre de usuario del administrador de WordPress.

Importante:
- esta será la cuenta administradora de tu WordPress
- no debe confundirse con la cuenta de usuario de Pantheon

#### Contraseña

Escribe y guarda una contraseña segura.

Importante:
- guarda bien esta contraseña
- la necesitarás para acceder al escritorio de WordPress

#### Tu correo electrónico

Escribe tu dirección de correo electrónico y comprueba que está bien escrita.

#### Visibilidad en los motores de búsqueda

WordPress permite marcar una opción para pedir a los motores de búsqueda que no indexen el sitio.

Para este tutorial, conviene indicar que no quieres que el sitio sea indexado, porque no se trata de una tienda real.

## Importante sobre las credenciales

Guarda muy bien:
- el nombre de usuario
- la contraseña

Estas credenciales pertenecen al administrador de WordPress.

No son las mismas que usas para entrar en Pantheon.

## Paso 3: finalizar la instalación

Cuando hayas completado el formulario, pulsa el botón final para continuar con la instalación.

## Paso 4: pantalla de confirmación

Después aparece una pantalla con un mensaje parecido a:

- `¡Lo lograste!`
- `WordPress ya está instalado. ¡Gracias, y que lo disfrutes!`

En esa misma pantalla se muestra:
- el nombre de usuario
- la contraseña elegida
- un enlace llamado `Acceder`

Haz clic en `Acceder`.

## Paso 5: iniciar sesión en WordPress

Se abrirá la pantalla de acceso de WordPress.

Ahí debes escribir:
- el nombre de usuario que acabas de crear
- la contraseña que acabas de definir

Después pulsa el botón para iniciar sesión.

## Resultado esperado

Al iniciar sesión correctamente, se abre el escritorio de WordPress.

## Advertencia práctica

Es importante no olvidar estas credenciales.

Según tu experiencia, en este tipo de entornos gratuitos o de prueba no siempre resulta fácil recuperar el acceso si se pierde la contraseña.

Por eso, si el usuario olvida estas credenciales, puede acabar teniendo que reinstalar el sitio y empezar de nuevo.
