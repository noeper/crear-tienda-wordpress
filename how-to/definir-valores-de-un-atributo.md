# How-to: definir los valores de un atributo

## Objetivo

Crear los valores de un atributo ya existente.

Ejemplo:
- atributo: Talla
- valores: S, M, L

## Requisitos previos

- Haber creado antes el atributo.

## Pasos

1. Ve a `Productos -> Atributos`.
2. En el atributo correspondiente, pulsa `Configure terms`.
3. Introduce el nombre del valor en `Name`.
4. Revisa o edita el `Slug` si es necesario.
5. Pulsa `Add new [nombre del atributo]`.
6. Repite el proceso para cada valor.

## Errores frecuentes

- Crear atributos duplicados como `color` y `colores`.
- Empezar los productos antes de definir bien los valores.
- No controlar el orden de los valores.
