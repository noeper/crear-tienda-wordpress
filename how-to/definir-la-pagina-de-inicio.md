# How-to: definir la página de inicio

## Objetivo

Configurar qué página se mostrará como página de inicio del sitio.

## Contexto

Por defecto, WordPress muestra la página de entradas del blog como inicio.

Después de instalar WooCommerce, puede interesar usar:
- la página `Shop`
- una página creada específicamente como `Home` o `Inicio`

## Ruta

`Escritorio de WordPress -> Ajustes -> Lectura`

## Pasos

1. Ve a `Escritorio de WordPress -> Ajustes -> Lectura`.
2. En la sección `Tu página de inicio muestra`, selecciona `Una página estática (seleccionar abajo)`.
3. En `Página de inicio`, elige la página que quieres usar como inicio.
4. Selecciona `Shop` si quieres que la tienda sea la portada.
5. O selecciona una página como `Home` o `Inicio` si has creado una portada propia.
6. Pulsa `Guardar cambios`.

## Resultado esperado

El sitio deja de mostrar por defecto las últimas entradas del blog y pasa a abrir en la página elegida como inicio.
