# How-to: instalar y activar WooCommerce

## Objetivo

Instalar y activar el plugin WooCommerce en WordPress.

## Requisitos previos

- Tener acceso al escritorio de WordPress.

## Ruta

`Escritorio de WordPress -> Plugins -> Añadir plugin`

## Pasos

1. Ve a `Escritorio de WordPress -> Plugins -> Añadir plugin`.
2. En la pantalla de plugins, localiza el campo `Buscar plugins`.
3. Escribe `WooCommerce`.
4. Revisa los resultados que aparecen.
5. Identifica el plugin correcto comprobando estos elementos:
   - el nombre `WooCommerce`
   - el icono con fondo violeta y el texto `woo` en minúscula
   - una descripción que indica que sirve para publicar una tienda online y crecer con ella
6. Si necesitas apoyo visual, puedes comparar el icono con:
   - `assets/images/icon-woocommerce.svg`
7. Pulsa `Instalar`.
8. Cuando termine la instalación, pulsa `Activar`.

## Resultado esperado

- WooCommerce queda instalado en el sitio.
- WooCommerce pasa a estar activo.
- En el panel lateral de WordPress aparecen las opciones propias de WooCommerce.

## Importante

No conviene identificar el plugin solo por una frase exacta de la descripción, porque ese texto puede cambiar con el tiempo.

Lo más seguro es confirmar a la vez:
- el nombre `WooCommerce`
- el icono violeta con `woo`
- y que el plugin corresponde a la herramienta de tienda online
