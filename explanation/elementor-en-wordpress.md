# Elementor en WordPress

## Qué es Elementor

Elementor es un plugin de WordPress orientado a la construcción visual de páginas.

Permite crear y editar páginas viendo directamente su aspecto, sin depender solo del editor clásico o del editor de bloques.

En lugar de trabajar únicamente con texto y configuraciones básicas, Elementor permite construir la página arrastrando y colocando elementos visuales.

## Para qué sirve

Elementor sirve para diseñar páginas con más control visual.

Por ejemplo, permite crear:
- páginas de inicio
- landing pages
- páginas de servicios
- páginas de contacto
- secciones promocionales dentro de una tienda

Su objetivo principal es facilitar la maquetación visual sin tener que programar desde cero.

## Cómo funciona a nivel general

Elementor trabaja a partir de bloques o elementos visuales que se colocan dentro de una página.

Entre esos elementos pueden estar:
- títulos
- textos
- imágenes
- botones
- columnas
- vídeos
- formularios

El usuario construye la página combinando estos elementos y ajustando su apariencia.

## Cómo es su interfaz

Elementor es un maquetador visual para WordPress que permite construir páginas trabajando directamente sobre su aspecto.

La interfaz se organiza, de forma general, en dos zonas principales:

- a la izquierda aparece un panel con bloques o elementos que se pueden usar en la página, como títulos, textos, imágenes, botones, columnas o secciones
- a la derecha aparece el lienzo de trabajo, donde se ve la página y donde se van colocando esos elementos

La forma de trabajo habitual consiste en arrastrar los bloques desde el panel izquierdo y soltarlos en el lienzo de la derecha.

Después, cada bloque puede editarse para cambiar su contenido, su estilo y su disposición.

## La opción Structure

Hay una opción especialmente importante en la cabecera, en la esquina superior izquierda: `Structure`.

Su icono es el de varias capas superpuestas.

`Structure` es una de las herramientas más útiles de Elementor porque permite ver la estructura interna de la página como un árbol de elementos.

Gracias a ella se puede identificar con claridad qué secciones, columnas, contenedores o bloques forman la página, aunque visualmente estén unos dentro de otros.

Esto es importante por varias razones:

- ayuda a entender cómo está construida la página realmente, no solo cómo se ve
- permite localizar más fácilmente un bloque cuando en el lienzo cuesta seleccionarlo
- facilita mover, reorganizar o borrar elementos sin equivocarse
- ayuda a detectar jerarquías, anidamientos y errores de composición

En la práctica, `Structure` es clave porque Elementor no consiste solo en arrastrar bloques visualmente: también exige entender cómo se organiza la página por dentro.

Por eso, aprender a usar bien `Structure` hace que trabajar con Elementor sea mucho más claro, más rápido y más preciso.

## Relación con WordPress

WordPress sigue siendo el sistema principal donde se gestionan:
- páginas
- entradas
- usuarios
- plugins
- temas

Elementor se añade sobre WordPress para ofrecer una forma distinta de editar el contenido visual de ciertas páginas.

Esto significa que:
- WordPress sigue siendo la base del sitio
- Elementor añade una capa de edición visual
- no sustituye por completo a WordPress, sino que amplía su forma de trabajo

## Relación con el tema

El tema sigue definiendo gran parte del aspecto general del sitio:
- cabecera
- pie de página
- estilos base
- estructura global

Elementor actúa sobre el diseño y composición de páginas concretas.

Por eso, tema y Elementor no son lo mismo:
- el tema define la base visual general
- Elementor ayuda a maquetar páginas concretas con más libertad

## Cuándo puede ser útil

Elementor puede ser útil cuando:
- se necesita crear páginas con una estructura visual más personalizada
- se quiere diseñar una landing page sin programar
- se busca rapidez para construir páginas promocionales

## Idea clave

Elementor es un plugin de construcción visual para WordPress.

Su función principal es facilitar el diseño de páginas mediante una interfaz visual, mientras WordPress sigue gestionando el contenido y la estructura general del sitio.
