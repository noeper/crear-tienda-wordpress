# Plugins en WordPress

## Qué es un plugin

Un plugin en WordPress es una extensión de software que añade nuevas funcionalidades al sitio sin modificar el núcleo de WordPress.

Permite ampliar lo que la web puede hacer.

Ejemplos:
- añadir una tienda online con WooCommerce
- crear formularios de contacto
- mejorar el SEO
- añadir copias de seguridad
- incorporar sistemas de seguridad

## Para qué sirven

Los plugins sirven para incorporar características que WordPress no incluye por defecto o para adaptar el sitio a necesidades concretas.

Por ejemplo, un sitio puede necesitar:
- vender productos
- mostrar un calendario de reservas
- traducir contenidos
- optimizar imágenes

En todos esos casos, un plugin permite añadir esa función sin tener que programarla desde cero.

## Relación con el núcleo de WordPress

WordPress incluye una base común de funciones. Los plugins se añaden sobre esa base.

Esto significa que:
- WordPress sigue siendo el sistema principal
- el plugin amplía o modifica su comportamiento
- varias funcionalidades del sitio pueden depender de plugins distintos

## Idea clave

Un plugin permite ampliar WordPress sin cambiar su núcleo.

Por eso, WordPress puede adaptarse a proyectos muy distintos usando una combinación de temas y plugins.
