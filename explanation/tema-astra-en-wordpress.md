# Astra en WordPress

## Qué es Astra

Astra es un tema de WordPress.

Su función principal es definir la base visual y estructural del sitio web.

Como cualquier otro tema, controla aspectos como:
- la apariencia general
- la cabecera
- el pie de página
- la disposición de los contenidos
- parte del estilo visual del sitio

## Para qué sirve

Astra sirve para dar al sitio una estructura visual lista para usar y fácil de adaptar.

Se utiliza con frecuencia porque permite construir sitios web de distintos tipos, por ejemplo:
- blogs
- webs corporativas
- tiendas online
- landing pages

Su objetivo es ofrecer una base de diseño flexible sobre la que después se puede trabajar con contenido, plugins y, en algunos casos, constructores visuales.

## Características principales

### Ligero y rápido

Astra está pensado para ofrecer buen rendimiento.

La idea general es que funcione como un tema ligero, con bajo consumo de recursos y una base razonablemente optimizada.

### Alta personalización

Permite modificar elementos como:
- cabecera
- pie de página
- tipografías
- colores
- layout

Todo esto se puede ajustar en gran parte sin necesidad de programar.

### Integración con constructores visuales

Astra está pensado para trabajar bien con herramientas como:
- Elementor
- Divi
- el editor de bloques de WordPress

Eso lo convierte en una base flexible para distintos flujos de trabajo.

### Compatibilidad con WooCommerce

Incluye soporte útil para tiendas online.

Puede influir en cómo se muestran:
- páginas de producto
- listados de productos
- carrito
- checkout

### Plantillas prediseñadas

Astra ofrece plantillas o sitios de partida que pueden servir como base para:
- webs corporativas
- tiendas online
- blogs

Esto permite empezar desde una estructura ya montada y adaptarla después.

## Relación con WordPress

WordPress gestiona el contenido del sitio:
- páginas
- entradas
- productos
- usuarios
- plugins

Astra actúa sobre cómo se presenta ese contenido.

Esto significa que:
- WordPress sigue siendo el sistema principal
- Astra define gran parte de la apariencia del sitio
- el contenido no depende de Astra para existir, pero sí puede verse distinto según el tema activo

## Relación con WooCommerce

Si el sitio usa WooCommerce, Astra influye en cómo se muestran elementos como:
- la tienda
- las fichas de producto
- las categorías de producto
- los listados de productos

Es decir, WooCommerce aporta la funcionalidad de tienda y Astra aporta buena parte de la presentación visual de esa tienda.

## Relación con plugins y constructores visuales

Astra no sustituye a los plugins.

Cada elemento tiene una función distinta:
- WordPress gestiona el sistema base
- el tema Astra define la apariencia general
- los plugins añaden funcionalidades
- un constructor visual como Elementor ayuda a diseñar páginas con más control visual

Por eso, Astra puede convivir con plugins de muchos tipos y también con herramientas de maquetación visual.

## Filosofía de Astra

Astra sigue una idea muy clara:

ser un tema base ligero sobre el que construir el diseño.

Eso implica que:
- no impone un diseño cerrado
- se adapta al contenido y a las herramientas del sitio
- funciona como una base flexible sobre la que construir

## Ventajas

- Es ligero y rápido.
- Es flexible.
- Puede usarse en proyectos de distintos tipos.
- Se integra bien con WooCommerce y con constructores visuales.

## Limitaciones

- Muchas opciones avanzadas dependen de versiones más completas del ecosistema Astra.
- Puede requerir otras herramientas para aprovechar todo su potencial visual.
- No define por sí solo un diseño fuerte si no se configura.

## Idea clave

Astra es un tema de WordPress.

Su papel principal es definir la base visual del sitio para que el contenido, los plugins y la tienda funcionen dentro de una presentación coherente.
