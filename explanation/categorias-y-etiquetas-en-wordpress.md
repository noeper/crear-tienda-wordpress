# Categorías y etiquetas en WordPress

## Qué son

Las entradas de WordPress pueden organizarse mediante:
- categorías
- etiquetas

Ambas sirven para clasificar contenido, pero no cumplen exactamente la misma función.

## Categorías

Las categorías son un sistema jerárquico de clasificación temática.

Sirven para organizar las entradas en secciones generales.

Ejemplo de estructura:
- Europa
  - Francia
  - Italia

Si una entrada habla de París, puede tener sentido clasificarla dentro de `Francia`, y `Francia` puede depender de `Europa`.

## Cómo conviene usarlas

En muchos proyectos se recomienda que cada entrada tenga una categoría principal clara, porque eso ayuda a mantener una arquitectura de contenidos más ordenada.

Sin embargo, conviene matizar algo importante:
- WordPress sí permite asignar varias categorías a una entrada.
- No es una limitación técnica usar solo una.
- Es una recomendación editorial o de organización en muchos casos, no una regla obligatoria del sistema.

Dicho de otro modo:
- si tu clasificación es única y bien pensada, suele tener sentido usar una categoría principal
- si el proyecto necesita dos clasificaciones distintas, una entrada puede pertenecer a varias categorías

Lo importante es diseñar la estructura con intención antes de empezar a publicar.

## Etiquetas

Las etiquetas son palabras clave o frases cortas que permiten agrupar entradas de forma más transversal.

Ejemplos:
- mochilero
- familiar
- romántico
- económico
- museos
- gastronomía

Características:
- no son jerárquicas
- una entrada puede tener varias
- ayudan a relacionar contenidos por temas concretos

## Diferencia práctica

Una forma útil de entenderlo es esta:
- las categorías ayudan a estructurar el sitio
- las etiquetas ayudan a describir o conectar contenidos

## Relación con WooCommerce

Cuando instalas WooCommerce, aparecen también categorías y etiquetas para productos.

La lógica conceptual es muy parecida:
- las categorías agrupan productos
- las etiquetas agrupan o describen productos de forma flexible

Pero no son exactamente las mismas taxonomías que las entradas del blog, ni están en el mismo sitio del panel de administración.
